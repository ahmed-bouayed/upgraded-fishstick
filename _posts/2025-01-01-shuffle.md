```shell
git clone https://github.com/Shuffle/Shuffle
cd Shuffle
```

```shell
mkdir shuffle-database
sudo chown -R 1000:1000 shuffle-database
sudo swapoff -a
```

add this line to `/etc/sysctl.conf`:

```shell
vm.max_map_count=262144
```

To apply the changes, run:

```shell
sudo sysctl -p
```

Run docker-compose

```shell
docker-compose up -d
```
